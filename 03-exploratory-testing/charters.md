# Charters

## Charter 1
**Título:** Exploración del flujo de compra

**Misión:** 
Explorar el proceso de compra en PetStore usando diferentes combinaciones de productos y acciones para identificar fallos en el carrito, validación de datos y flujo de checkout.

**Área principal explorada:** 
Carrito de compras y checkout


## Charter 2
**Título:** Exploración del manejo de carrito

**Misión:** 
Explorar el comportamiento del carrito al agregar, eliminar y modificar productos para detectar inconsistencias, pérdida de datos o errores de estado.

**Área principal explorada:** 
Carrito de compras
